    1. Primeros pasos

    Para empezar nuestro documento HTML 5, tenemos que indicar que tipo de documento es,esto se indica con < !DOCTYPE html >
    seguido del lenguaje que vayamos a usar en nuestra página. < html lang= "es" > (Español)
    Para cerrar nuestras etiquetas utilizamos su mismo nombre con una barra, tal que así. < /html >

    Después debemos indicar el título que va a tener nuestro documento dentro de la cabecera.
    Para iniciar la cabecera < head > y sin cerrar la etiqueta, ponemos dentro nuestro título < title >< /title >
    también señalamos dos etiquetas más que contienen el código UNICODE de nuestra página < meta charset = "utf-8" >
    y el nombre del autor de la página < meta name = "author" content "Javier" > y ya tenemos una buena cabecera para poder empezar con el cuerpo del documento.

    Con la etiqueta < body > abrimos el cuerpo del documento, aqui se pueden introducir diversas etiquetas según lo que quieras hacer
    Si queremos poner títulos a nuestros documentos basta con la etiqueta < h1 > Título dentro de nuestra etiqueta < body >
    el tamaño es configurable añadiendo +1 a nuestra etiqueta < h2 >< /h2 >,< h3 >< /h3 >...hasta 6 tamaños diferentes, siendo 1 el mayor y 6 el más pequeño.
    Un primer documento de prueba quedaría con la siguiente estructura.

    		
    
												
    
													
				< !DOCTYPE html >
				< html lang=“es” >
					< head >
						< title >Primer documento HTML< /title >
						< meta charset=“utf-8” >
						< meta name=“author” content=“Javier” >
					< /head > 
					< body >
						< h1 > Mi primer HTML < h1 >
					< /body >
					< /html >

    2. Artículos y Párrafos

    Los artículos cuentan con secciones donde puedes separar la información escrita en distintos bloques para que no se vea tan sobrecargado.
    El formato para crear un artículo es el siguiente:

    	
    
										
    
											
				< article >

					< section >
					< /section >
					< section >
					< /section >

				< article >

    Existe la opción de crear diferentes párrafos para el manejo de la información, pueden crearse con < p > y al terminar < /p >

    Este texto esta en un párrafo y vamos a ver claramente la separación entre este parrafo y el siguiente.

    Este texto esta en otro párrafo.

    Los saltos de línea, espacios y tabulaciones también son configurables, con la etiqueta < br > hacemos un salto de línea tal y como hago (literalmente)
    a continuación.
    Pero también para las tabulaciones podemos poner una etiqueta de prediseño < pre > hasta que terminemos de tabular.

    3. Listas

    Las listas tienen mas opciones que los artículos, hay una variedad de listas y tienen distintos formatos, los elementos de una lista se señalan con < li >
    y pueden ser desordenadas < ul > , ordenadas < ol > o por definición en esta ultima las etiquetas varian, se escribe < dl >, < dt > para los términos y < dd > para el significado.

		< li > 1 < /li >			< li > 1 < /li >				< dt >hola:
		< li > 2 < /li >			< li > 2 < /li >					< dd > Forma de saludo. < /dd >
		< li > 3 < /li >			< li > 3 < /li >				< /dt >
	< /ul >					< /ol >						< /dl >

    5. Tablas

    para las casillas tenemos dos etiquetas una son las columnas < tr > y otra las filas < td >. 
    
							
    
										
			< table border=1 >
				< tr >

    6. URL

    Los Hiperenlaces pueden hacerse de dos tipos, apuntando hacia fuera del documento o dentro del mismo, para crear hiperenlaces apuntando hacia fuera del documento
    es necesaria la etiqueta < a href="">, para darle un nombre a nuestro enlace después ponemos el nombre de nuestro enlace acompañado de , así quedaría el ejemplo.
    
									
    
										
		< a href="dirección que queramos">Esto es un hiperenlace< /a >
    Para crear un enlace al mismo documento tenemos que crear dos etiquetas, una es donde vamos a ir < a name="Prueba" >< /a >
    y otra es desde donde vamos a ir< a href="#Prueba">Ir a Prueba< /a >, estos dos valores son llamados "Ancla" y "Enlace". 
    
									
    
										
			Ancla:
			 < a name="Prueba" >< /a >
			
    
			Enlace a Prueba:
			< a href="#Prueba">Ir a Prueba< /a >
    La creacion de imágenes es muy parecida a los hiperenlaces ya que los dos necesitan de una URL, para las imágenes la etiqueta < img src = "Mi imagen">
    también se puede editar el ancho y el largo de la imagen, asi como un comentario descriptivo de la foto. 
    
									
    
										
			< img src = "URL_de_mi_imagen" >
			< width = 200 >
			< height = 150 >
			< alt = "Esta es mi imagen"
    Hay una gran variedad de formatos de imágenes, Bimap,JPG,GIF,PDF,etc..Incluso hacer que una imagen nos lleve a otra URL, tenemos distintas zonas
    donde colocar los enlaces, pero existen formas (círculo,cuadrado,triángulo) para poder poner muchos enlaces en una sola imagen según donde hagas "click" con el ratón. 
    
									
    
										
		
			< img usermap="#URL_de_mi_imagen" >
			< map name = "Imagen" >
			< area shape "circle" coords= "(40,35)","(15,25)">
